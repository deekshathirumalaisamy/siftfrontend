📊 Engagement Metrics

Engagement Metrics module analyzes campaign performance by tracking engagement values, performance indicators and anomaly detection.

About Metric Management

This module stores and monitors important campaign metrics such as engagement scores, metric types, numeric values and unusual activity. It helps brands understand influencer campaign effectiveness.

Features

  • Track campaign engagement metrics
  • Store different metric types
  • Analyze numeric performance values
  • Detect abnormal engagement patterns
  • Monitor campaign analytics
  • Generate performance insights

Workflow

Campaign activity collected → Metrics generated → Performance analyzed → Anomalies detected → Reports improved.
